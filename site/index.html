Aqui está um exemplo de um portfólio profissional em HTML que atende aos requisitos que você mencionou. O código inclui um header fixo com um menu hambúrguer, uma seção Hero com um layout de grid 50/50, uma seção de Projetos com 3 cards e um formulário de contato válido. Além disso, inclui links para CSS e JS. ### Estrutura de Arquivos Para este exemplo, você pode criar os seguintes arquivos: - `index.html` - `styles.css` - `script.js` ### Código HTML (`index.html`) ```html Portfólio Profissional

``` ### Código CSS (`styles.css`) ```css body { font-family: Arial, sans-serif; margin: 0; padding: 0; } header { position: fixed; top: 0; left: 0; right: 0; background-color: #333; color: white; padding: 10px; display: flex; justify-content: space-between; align-items: center; } .menu { display: flex; align-items: center; } .hamburger { cursor: pointer; font-size: 24px; } nav { display: none; } nav ul { list-style-type: none; padding: 0; } nav ul li { display: inline; margin: 0 10px; } nav ul li a { color: white; text-decoration: none; } .hidden { display: none; } .hero { display: flex; justify-content: space-between; align-items: center; padding: 100px 20px; background-color: #f4f4f4; } .hero-content { flex: 1; } .hero-image { flex: 1; } .projects { padding: 50px 20px; } .project-cards { display: flex; justify-content: space-between; } .card { border: 1px solid #ccc; padding: 20px; flex: 1; margin: 0 10px; } .contact { padding: 50px 20px; } form { display: flex; flex-direction: column; } form label { margin: 10px 0 5px; } form input, form textarea { padding: 10px; margin-bottom: 10px; } form button { padding: 10px; background-color: #333; color: white; border: none; cursor: pointer; } ``` ### Código JavaScript (`script.js`) ```javascript function toggleMenu() { const navbar = document.getElementById('navbar'); navbar.classList.toggle('hidden'); } ``` ### Explicação 1. **Header Fixo com Menu Hambúrguer**: O header é fixo no topo da página e contém um menu que pode ser mostrado ou escondido ao clicar no ícone de hambúrguer. 2. **Seção Hero**: A seção hero é dividida em duas partes, uma para o conteúdo e outra para a imagem, utilizando flexbox para o layout 50/50. 3. **Seção Projetos**: Contém três cards que podem ser preenchidos com informações sobre os projetos. 4. **Formulário de Contato**: Um formulário simples que inclui validação básica para os campos obrigatórios. 5. **Links para CSS/JS**: O HTML inclui links para os arquivos CSS e JavaScript. Você pode personalizar ainda mais o estilo e o conteúdo conforme necessário!
